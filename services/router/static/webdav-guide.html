WebDAV adapter

Connect with WebDAV

The WebDAV endpoint is meant for WebDAV clients and desktop file managers, not direct browser browsing. Use it to mount journal-backed documents as files.

Authentication

Use an API token as the password for Basic authentication. The username is accepted by most clients but is not the secret; the token/password is what authenticates the request.

Useful mount URLs

Mutable staged documents /webdav/stage/
Committed latest ledger state /webdav/ledger/state/
Pin/unpin control sink /webdav/control/pin
Explicit snapshots and bridges /webdav/ledger/…

Desktop clients

  • GNOME Files / Nautilus: connect to dav://HOST/webdav/stage/ or davs://HOST/webdav/stage/.
  • macOS Finder: use Go → Connect to Server… and enter http://HOST/webdav/stage/ or https://HOST/webdav/stage/.
  • Windows: use Map network drive or Add a network location with the HTTP(S) WebDAV URL.

Command-line smoke test

Replace HOST and API_TOKEN, then write and read a small file through the staged projection:

printf 'hello journal\n' > /tmp/sync-web-webdav.txt
curl -u "user:API_TOKEN" -T /tmp/sync-web-webdav.txt \
  http://HOST/webdav/stage/admin/hello.txt
curl -u "user:API_TOKEN" \
  http://HOST/webdav/stage/admin/hello.txt
